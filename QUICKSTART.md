# Quickstart

## 1. 通用舆情数据处理

```text
请使用 $public-opinion-data-workflow 处理这个表格：
1. 检查工作表、字段和行数；
2. 判断正文、平台、时间、链接、用户、位置等字段；
3. 先做隐私审计和脱敏建议；
4. 清洗、去重、筛选；
5. 输出每一步数量、处理明细、删除记录和统计表。
```

## 2. 大规模评论情感/态度分析

```text
请使用 $public-opinion-data-workflow 处理这个中文评论数据集。
这是大样本情感/态度分析任务，不要默认全量 AI 逐条标注。
请先清洗脱敏，确认或建立标签口径，再抽样 AI 标注，训练/校准分类器，迁移到全量，抽低置信、边界、反讽和随机样本复核，最后输出统计、图表和质量摘要。
```

## 3. 没有现成情感标签时

```text
这个新议题还没有确定情感/态度分类。
请先抽样查看评论表达，必要时用聚类/关键词发现常见表达簇，再提出候选标签体系和判断标准。
等我确认标签口径后，再进入抽样 AI 标注、分类器迁移和复核流程。
```

## 4. 有历史标签可对照时

```text
表里有一个历史标签列：话题_v25。
请把它作为参考标签，与新输出的标签列做对照，输出一致率、混淆矩阵和差异样本。
注意：历史标签只是参考，不一定是绝对真值。
```
